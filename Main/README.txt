[タイトル画面での操作方法]
↑,↓     … 選択
スペース  … 決定

[ゲーム中の操作方法]
W         … ハードドロップ
S         … ソフトドロップ
A, D      … 左右移動
J, K      … 左右回転
スペース  … ホールド

[このゲームは]
　 降ってくる「ミノ」を移動させたり回転させたりさせて積み重ねていくゲームです。
 横１列をブロックで埋めるとブロックは消えて、スコアが加算されます。

 ブロックが画面の一番上まで積み重なってしまうとゲームオーバーです。

 ５分の制限時間がついています。制限時間内にできるだけ多くのスコアを稼ぎましょう。

[ミノについて]
  　ミノは４つのブロックで作られています。
  ミノは７つ種類があり、それぞれの形はアルファベットの「T, I, L, J, S, Z, O」に例えられます。

[ランキング機能]
 　ゲーム終了後、リザルト画面でスペースを押すと自分のスコアをランキングに登録できます。
 ランキングは工嶺祭期間中に限定でアクセスできるサイトで確認することができます。

[ルール詳細]
   ・ホールド
        　スペースキーを押すとホールドをすることができます。
　　　　ホールドをすると画面左上の「HOLD」の部分のミノと、
        落下しているミノを入れ替えることができます。

　　　　１度ホールドをすると、降ってきているミノを積むまではホールドできません。

	　ゲームが始まってから最初にするホールドは、落下中のミノが「HOLD」に入り、
　　　　次のミノが降ってきます。

   ・Level
        　Levelは画面下部に表示されます。
        一定のスコアを超えるとレベルが上がります。
        レベルが上がるとミノが自然に落ちてくるスピードが速くなります。

   ・Ren
        　ブロックを連続で消すと「Ren」が発生します。
        Renが発生すると通常よりも多くスコアが加算されます。

        ３回、４回と連続でブロックを消していくとRenの数も増えていき、
        スコアも多く加算されるようになります。

[テクニック]
 　ミノは地面に付いてからでも左右に動かしたり、回転させたりすることで少しのあいだ時間を稼ぐことができます。
